Mobile Security & Privacy Simulation
  • Mobile Security & Privacy Simulator is SimPy-based simulation framework for the high-level evaluation of mobile security and privacy scenarios. It was created in the context of a student project at the Leibniz Universität Hannover. It was enhanced and used within research.
    (advisor, enhancing contributor)
  • MoSP Geo Tool is a PyGTK-based editor for OpenStreetMap maps: Generalize maps, connect partitions, connect POIs with the road network, aso. for using with the MoSP simulator. It was created in the context of a bachelor thesis at Leibniz Universität Hannover.
    (advisor)
  • MoSP Siafu is an extension of the Siafu Open Source Context Simulator created for indoor simulation when running multiple coupled simulations based on the MoSP simulator. It was created in the context of a student project at Leibniz Universität Hannover.
    (advisor)
